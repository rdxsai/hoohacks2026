# Consumer & Prices Sector Agent — Tool Inventory

## Input (Provided by Orchestrator, Not a Tool)

The agent receives:
1. **Policy Analyst's Briefing** — policy spec, baseline data,
   transmission channels, sector exposure matrix
2. **Labor Agent's Output** (if available) — wage change estimates,
   employment effects, benefits cliff data
3. **Housing Agent's Output** (if available) — rent change estimates,
   housing cost changes by region

These are INPUTS to the Consumer Agent's analysis. The agent uses
its own tools for price-specific deep research.

## Tool 1: bls_get_data

Consumer Price Index — the definitive measure of what consumers pay.
Also produces the Producer Price Index (upstream price pressure) and
Consumer Expenditure Survey data (how households spend). This tool
covers three critical data programs for this agent.

| CUSR0000SEFV | CPI Food away from home (restaurants) | Restaurant price impact |
| CUSR0000SAH1 | CPI Shelter | Housing costs (coordinate with Housing Agent) |

**PPI Series (Upstream Price Pressure) — Producer Price Index:**

## Tool 4: census_acs_query

**Purpose in Consumer Context:** Household income distribution at
county/metro level. This is how you know what fraction of households
in each area fall into each income tier, which is essential for the
distributional analysis. Also provides poverty rates, SNAP/Medicaid
enrollment proxies, and household composition data.

**Key ACS Tables for Consumer Analysis:**

| Table | What It Measures |
|-------|-----------------|
| B19001 | Household income distribution (brackets) | 
| B19013 | Median household income |
| B19025 | Aggregate household income |
| B19301 | Per capita income |
| B17001 | Poverty status by age and sex |
| B22003 | SNAP/food stamp receipt |
| B11001 | Household type and size |
| B08301 | Means of transportation to work (for transport cost analysis) |
| B27010 | Health insurance type (for healthcare cost analysis) |
| S1901 | Income summary table (has mean, median, and distribution) |

**Why this matters for consumer analysis:**
You need to know how many households are in each income tier to
weight your impact scorecards properly. If 40% of households in
Southwest Virginia earn below $40K, and your analysis shows those
households lose $45/month in purchasing power, that's a much bigger
story than if only 10% are in that tier.

**API:** Census — `https://api.census.gov/data/{year}/acs/acs5`
**Auth:** Census API key (free)
**Cost:** Free

## Tool 5: bea_regional_data

**Purpose in Consumer Context:** Regional Price Parities (RPP) and
Personal Consumption Expenditure by state. RPPs tell you how much
more or less expensive each region is compared to the national
average — critical for translating national price change estimates
into local dollar impacts.

**Key BEA Tables:**

| Table | What It Measures | Use For |
|-------|-----------------|---------|
| SARPP | State Regional Price Parities (overall) | State cost-of-living comparison |
| MARPP | Metro Regional Price Parities | Metro cost-of-living comparison |
| SAPCE1 | Personal consumption by state | State spending patterns |
| CAINC1 | Personal income by county | County income baseline |
| SAINC7 | Wages and salaries by industry by state | Income composition |

**Regional Price Parities breakdown:**
- RPP (Goods) — tradeable goods price level
- RPP (Services: Rents) — housing cost level
- RPP (Services: Other) — non-housing services cost level
- RPP (All items) — overall cost of living

**Virginia RPP context (approximate):**
- Northern Virginia (DC metro): RPP ~115-120 (15-20% above national)
- Richmond metro: RPP ~95-100 (near national average)
- Hampton Roads: RPP ~90-95 (slightly below)
- Rural/Southwest VA: RPP ~80-85 (15-20% below national)

This means a $50/month price increase nationally translates to
~$57-60 in NoVA and ~$40-42 in rural VA.

**API:** BEA — `https://apps.bea.gov/api/data/`
**Auth:** BEA API key (free)
**Cost:** Free

## Tool 11: usda_food_prices

**Purpose in Consumer Context:** USDA Economic Research Service
tracks food prices with more granularity than BLS CPI. Provides
food price outlook, farm-to-retail price spreads, and food
expenditure data broken down by income level and food category.

**Key USDA ERS Datasets:**
- Food Price Outlook — monthly and annual food price forecasts
- Food Expenditure Series — food spending by source and food category
- Quarterly Food-at-Home Price Database — item-level grocery prices
- Food Access Research Atlas — food desert mapping

**Access method:** USDA ERS does not have a real-time REST API.
Data is published as downloadable datasets and reports at
`ers.usda.gov/data-products/`. Use `web_search_news` to find
the most current USDA food price outlook report, then
`fetch_document_text` to read it.

**Workaround implementation:**
```python
# Search for latest USDA food price data
web_search_news("USDA ERS food price outlook 2026", recency="past_month")

# Then fetch the found report
fetch_document_text("https://ers.usda.gov/data-products/food-price-outlook/")
```

**Cost:** Free (public data)

## Tools NOT Included (Rationale)

| Tool Considered | Why Excluded |
|----------------|--------------|
| Nielsen/IRI grocery scanner data | Paid commercial product, $$$$ |
| Credit card transaction data (Visa/Mastercard aggregates) | Not publicly available as API |
| GasBuddy API | Niche (gas prices only), web_search_news covers this |
| Numbeo cost of living API | User-submitted data, unreliable for policy analysis |
| MIT Billion Prices Project | Discontinued/absorbed into academic work |
| Yelp Fusion API (restaurant prices) | No historical price data, not useful for policy analysis |
