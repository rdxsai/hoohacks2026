# Housing & Cost of Living Agent — Tool Inventory

## Input (Not a Tool — Provided by Orchestrator)

The agent receives the **Policy Analyst's Briefing** as structured
context before any tool calls. This provides:
- Policy specification (exact parameters, scope, phase-in)
- Baseline economic data (income levels, employment, CPI)
- Transmission channel map (which channels affect housing)
- Sector exposure matrix (housing row gives starting direction)

The agent uses this as its STARTING POINT, not as sufficient input.
All tools below are for the agent's OWN deep research.

## Tool 1: fred_search

**Purpose in Housing Context:** Discover housing-related FRED series
the agent doesn't know the ID for. Housing data in FRED is extensive
but series IDs are not intuitive.

**Typical Housing Queries:**
- "housing starts" → HOUST
- "building permits" → PERMIT
- "30 year mortgage rate" → MORTGAGE30US
- "median home price" → MSPUS
- "Case Shiller home price index" → CSUSHPINSA
- "rental vacancy rate" → RRVRUSQ156N
- "homeownership rate" → RHORUSQ156N
- "house price index virginia" → state-level HPI
- "real residential investment" → PRRESCONS
- "construction spending residential" → TLRESCONS

**API:** FRED — `https://api.stlouisfed.org/fred/series/search`
**Auth:** FRED API key (free)
**Cost:** Free

## Tool 2: fred_get_series

**Purpose in Housing Context:** Pull actual time series data for
housing market indicators. This is the primary source for national
and state-level housing market conditions.

**Critical Housing Series:**

| Series ID | What It Measures | Use For |
|-----------|-----------------|---------|
| MORTGAGE30US | 30-year fixed mortgage rate | Mortgage affordability |
| MORTGAGE15US | 15-year fixed mortgage rate | Mortgage affordability |
| HOUST | Housing starts (national) | Supply pipeline |
| PERMIT | Building permits (national) | Supply pipeline leading indicator |
| MSPUS | Median sales price of houses sold | Home price baseline |
| CSUSHPINSA | Case-Shiller Home Price Index (national) | Home price trend |
| RRVRUSQ156N | Rental vacancy rate | Market tightness |
| RHORUSQ156N | Homeownership rate | Tenure composition |
| TLRESCONS | Total residential construction spending | Construction activity |
| CUSR0000SEHA | CPI - Rent of primary residence | Rent inflation |
| CUSR0000SEHC | CPI - Owners' equivalent rent | Imputed owner cost |
| TTLHHM[ST]A156N | Total households by state | Household formation |
| [STATE]HPINDEX | State-level house price index | Regional price trends |
| FIXHAI | Housing Affordability Index | Composite affordability |

**State-specific patterns:**
- `[STATE]POP` — State population
- `[STATE]HPINDEX` or `ATNHPIUS[FIPS]A` — State/metro HPI
- FRED has limited state-level rent data; use Census ACS for this

**API:** FRED — `https://api.stlouisfed.org/fred/series` + `/observations`
**Auth:** FRED API key (free)
**Cost:** Free

## Tool 4: census_acs_query

**Purpose in Housing Context:** THE critical tool for local housing
data. Census ACS provides rent levels, home values, housing cost
burden, income distribution, and housing characteristics at county
and metro area level. No other source gives you this geographic
granularity for housing affordability.

**Key ACS Tables for Housing:**

| Table | What It Measures | Geography |
|-------|-----------------|-----------|
| B25064 | Median gross rent | County, metro, tract |
| B25077 | Median home value | County, metro, tract |
| B25070 | Gross rent as % of household income (distribution) | County, metro |
| B25091 | Mortgage costs as % of household income | County, metro |
| B25001 | Total housing units | County, metro, tract |
| B25002 | Occupancy status (occupied vs vacant) | County, metro |
| B25003 | Tenure (owner vs renter) | County, metro |
| B25014 | Tenure by persons per room (overcrowding) | County, metro |
| B19013 | Median household income | County, metro, tract |
| B25063 | Gross rent (distribution by dollar amount) | County, metro |
| B25075 | Home value (distribution by dollar amount) | County, metro |
| DP04 | Housing characteristics profile | County, metro |

**Virginia FIPS codes you'll need:**
- 51 = Virginia (state)
- 51013 = Arlington County
- 51059 = Fairfax County
- 51760 = Richmond City
- 51810 = Virginia Beach City
- 51770 = Roanoke City
- 47900 = Washington-Arlington-Alexandria MSA
- 40060 = Richmond MSA
- 47260 = Virginia Beach-Norfolk MSA

**API:** Census — `https://api.census.gov/data/{year}/acs/acs5`
**Auth:** Census API key (free, instant registration)
**Cost:** Free

**Implementation:**
```python
async def census_acs_query(
    table_variables: list[str],
    geography: str,  # "county:*" or "county:013" or "metropolitan statistical area/micropolitan statistical area:47900"
    state_fips: str = "51",  # Virginia
    year: int = 2023,
    dataset: str = "acs5"  # acs1 for areas >65K pop, acs5 for all
) -> str:
    base_url = f"https://api.census.gov/data/{year}/acs/{dataset}"
    params = {
        "get": ",".join(table_variables),
        "for": geography,
        "in": f"state:{state_fips}",
        "key": CENSUS_API_KEY,
    }
    resp = await httpx.AsyncClient().get(base_url, params=params)
    return json.dumps(resp.json())
```

## Tool 5: bea_regional_data

**Purpose in Housing Context:** Regional price parities (RPP) —
tells you how much more/less expensive each metro area is compared
to the national average. Critical for comparing real (cost-adjusted)
income across regions and understanding geographic affordability
variation.

**Key BEA Tables for Housing:**

| Table | What It Measures |
|-------|-----------------|
| CAINC1 | Personal income by county |
| CAINC4 | Personal income and employment by county (detailed) |
| SARPP | State-level regional price parities |
| MARPP | Metro-level regional price parities |
| CAGDP2 | GDP by industry by county (identifies construction sector share) |

**Regional Price Parities are uniquely valuable:**
- RPP > 100 means more expensive than national average
- RPP < 100 means cheaper
- RPP for "rents" specifically tells you housing cost relative to nation
- Northern Virginia RPP (rents) ≈ 130-140 (30-40% above national avg)
- Rural Virginia RPP (rents) ≈ 70-80 (20-30% below)

**API:** BEA — `https://apps.bea.gov/api/data/`
**Auth:** BEA API key (free registration)
**Cost:** Free

**Implementation:**
```python
async def bea_regional_data(
    table_name: str,  # "CAINC1", "SARPP", "MARPP", "CAGDP2"
    geo_fips: str,    # "51000" for VA state, "51013" for Arlington
    year: str = "2023",
    line_code: str = "1"  # varies by table
) -> str:
    params = {
        "UserID": BEA_API_KEY,
        "method": "GetData",
        "datasetname": "Regional",
        "TableName": table_name,
        "GeoFIPS": geo_fips,
        "Year": year,
        "LineCode": line_code,
        "ResultFormat": "JSON",
    }
    resp = await httpx.AsyncClient().get(
        "https://apps.bea.gov/api/data/", params=params
    )
    return json.dumps(resp.json()["BEAAPI"]["Results"]["Data"])
```

## Tool 6: hud_data

**Purpose in Housing Context:** HUD Fair Market Rents (FMR) are the
official benchmark rents used by the federal government for housing
vouchers and affordability calculations. Also provides income limits
(what counts as "low income" in each area) and housing affordability
indices.

**Key HUD Datasets:**

| Dataset | What It Measures | Update Freq |
|---------|-----------------|-------------|
| Fair Market Rents (FMR) | 40th percentile rent by bedroom count by county/metro | Annual |
| Income Limits | Low/very-low/extremely-low income thresholds by area | Annual |
| USPS Vacancy Data | Residential vacancy rates by county (from mail delivery data) | Quarterly |
| Picture of Subsidized Households | Subsidized housing inventory and characteristics | Annual |

**API:** HUD User — `https://www.huduser.gov/hudapi/public/`
**Auth:** Token from huduser.gov (free registration)
**Cost:** Free

**Endpoints:**
| Endpoint | Returns |
|----------|---------|
| `/fmr/statedata/{state}` | FMRs for all counties in state |
| `/fmr/data/{entity_id}` | FMR for specific county/metro |
| `/il/statedata/{state}` | Income limits for all counties in state |
| `/il/data/{entity_id}` | Income limits for specific county/metro |
| `/usps/{entity_id}?year={y}&quarter={q}` | Vacancy data |

**Implementation:**
```python
async def hud_data(
    dataset: str,    # "fmr", "il", "usps"
    entity_id: str,  # FIPS code or "VA" for state-level
    year: int = 2025,
    quarter: int = None  # only for USPS vacancy data
) -> str:
    base = "https://www.huduser.gov/hudapi/public"
    
    if dataset == "fmr":
        if len(entity_id) == 2:  # state abbreviation
            url = f"{base}/fmr/statedata/{entity_id}"
        else:
            url = f"{base}/fmr/data/{entity_id}"
    elif dataset == "il":
        if len(entity_id) == 2:
            url = f"{base}/il/statedata/{entity_id}"
        else:
            url = f"{base}/il/data/{entity_id}"
    elif dataset == "usps":
        url = f"{base}/usps/{entity_id}"
        if year and quarter:
            url += f"?year={year}&quarter={quarter}"
    
    headers = {"Authorization": f"Bearer {HUD_TOKEN}"}
    resp = await httpx.AsyncClient().get(url, headers=headers)
    return json.dumps(resp.json())
```

## Tools NOT Included (Rationale)

| Tool Considered | Why Excluded |
|----------------|--------------|
| Zillow API | Zillow killed their free API in 2021. Data accessible via web_search_news instead. |
| Redfin API | No public API. Data accessible via web_search_news. |
| Realtor.com API | Requires partnership agreement. Not feasible for hackathon. |
| CoreLogic | Paid enterprise product ($$$). Academic-only access. |
| CoStar / Apartments.com | Paid commercial product. Not accessible. |
| NMHC Rent Survey | Members only. Key findings surface via web_search_news. |

The free government data stack (Census + HUD + FRED + BLS + BEA) covers
roughly 85% of what these paid sources provide for policy analysis
purposes. The paid sources add granularity (zip-code level, monthly
frequency, individual property data) that matters for real estate
investment but is unnecessary for policy impact analysis.
